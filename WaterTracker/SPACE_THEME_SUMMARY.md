## 技术实现

### CSS 技术
- **box-shadow** 创建星星粒子
- **radial-gradient** 创建星云和行星效果
- **conic-gradient** 创建旋转光圈
- **backdrop-filter: blur()** 玻璃质感
- **CSS animations** 所有动画效果
- **多层 box-shadow** 发光效果

### 性能优化
- 使用 `transform` 和 `opacity` 进行动画（GPU加速）
- 避免使用昂贵的 `filter` 动画
- 使用 `pointer-events: none` 避免伪元素干扰交互
